# linguafill
Multilingual Voice-Assisted Form Filling System using  Flask, Python, OCR, NLP 
Developed an AI-powered form automation system supporting 3 languages using OCR, speech recognition, and
NLP.
 Automated extraction and mapping of 20+ form fields from uploaded documents using Tesseract OCR.
Improved form-filling accuracy to 88% through intelligent validation and multilingual voice interaction.
